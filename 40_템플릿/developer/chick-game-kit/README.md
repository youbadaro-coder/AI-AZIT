# 🐤 병아리게임 샘플팩 — 다마고치 스타일 미니 게임

다마고치 스타일 단일 HTML 미니 게임. 프레임워크·빌드 없음. 클릭 한 번이면 작동.

## 언제 쓰나
- 코딩 처음 배우는 사람의 첫 결과물 데모
- 라이브 시연용 "5초 안에 결과 보이는" 미니 프로젝트
- vanilla HTML/CSS/JS 학습 교재
- 캐주얼 한 페이지 인터랙티브 데모

## 게임 구성
- **3개 스탯**: 배고픔·행복·에너지 (자동 감소)
- **4개 액션**: 먹이주기·놀아주기·재우기·씻기기
- **성장 시스템**: 0일 아기 → 5일 어린이 → 15일 청소년 → 30일 어른닭
- **반응형**: 모바일에서 4 버튼이 2×2 그리드로
- **터치 가능**: 병아리 클릭하면 쓰다듬기 + 하트 애니메이션

## 사용법

### 1. 새 프로젝트 폴더 만들기
```
~/connect-ai-projects/chick-game/
```

### 2. 이 폴더의 index.html 을 프로젝트 루트로 복사
모든 코드는 단일 파일 안에 (style 태그·script 태그 포함).

### 3. 브라우저로 열기
```bash
open index.html
```

또는 로컬 서버:
```bash
cd ~/connect-ai-projects/chick-game
python3 -m http.server 8000
# http://localhost:8000 접속
```

## 커스터마이즈 포인트
- **색상 팔레트**: `<style>` 상단의 `background` 그라데이션 (현재: 하늘색 → 잔디색)
- **병아리 SVG**: `<svg id="chickSvg">` 부분 — 색상·모양 자유 변경
- **메시지 톤**: `const messages = { ... }` — 한국어 톤 자유 교체
- **감소 속도**: `setInterval(... 5000)` — 5초 → 다른 값으로 난이도 조정
- **성장 단계 추가**: `update()` 함수 안 stage 로직에 새 단계 추가

## 응용 아이디어
- 강아지/고양이 키우기로 SVG·메시지만 바꾸면 즉시 다른 게임
- localStorage 추가로 새로고침해도 상태 유지
- 사운드 효과 추가 (Web Audio API)
- 친구 병아리 공유 (URL 파라미터 / Firebase)
- 미니게임 추가 (병아리 먹이 캐치 등)

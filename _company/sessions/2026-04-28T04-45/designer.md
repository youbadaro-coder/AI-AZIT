## 📊 비주얼 에셋 최종 검토 및 개발 스펙 정리

CEO의 지시에 따라 현재 승인된 모든 비주얼 에셋(썸네일, 브랜드 가이드)을 최종 확인하고, 개발팀이 즉시 구현할 수 있도록 **구체적이고 실행 가능한 디자인 스펙**을 정리합니다.

### 1. 브랜드 디자인 시스템 최종 검증 (Brand Visual System Validation)

모든 요소가 '회사 정체성'과 'CTR 최적화 전략'에 부합하는지 최종 점검했습니다.

*   **컬러 팔레트:**
    *   **Primary Color (핵심):** `#FF4500` (강렬한 액션/주의 유도) - 사용 범위: CTA 버튼, 핵심 텍스트 강조.
    *   **Secondary Color (보조):** `#1A1A1A` (딥 블랙) - 사용 범위: 배경, 주요 정보 영역의 대비 강화.
    *   **Accent Color (강조):** `#00FFFF` (시안/미래 지향성) - 사용 범위: 포인트 아이콘, 그래프 강조선.
    *   **Background Color:** `#F5F5F5` (밝은 그레이) - 사용 범위: 콘텐츠 배경으로 안정감 부여.

*   **타이포그래피 (Typography):**
    *   **Headline Font:** **'Montserrat Bold'** (또는 유사한 산세리프 계열) - **용도:** 썸네일 메인 헤드라인. (가독성과 강렬함 동시 확보)
    *   **Body Font:** **'Noto Sans KR'** 또는 시스템 기본 폰트 - **용도:** 영상 내 설명 텍스트, 상세 정보 표기. (한국어 가독성 최우선)
    *   **스타일 규칙:** 헤드라인은 굵게 처리하고, 핵심 키워드는 Primary Color(`FF4500`)로 오버레이하여 시선을 즉각적으로 사로잡아야 함.

*   **레퍼런스 및 스타일 가이드 (Reference & Style Guide):**
    *   **썸네일 레이아웃:** **'정보 밀도(Information Density) 극대화'** 원칙 적용.
        1.  **좌측/우측 분할:** 핵심 이미지/인물(감성 자극) + 텍스트 블록(명확한 가치 제안).
        2.  **색상 대비:** 배경과 텍스트 간의 명도 차이를 최대화하여 모바일 환경에서도 3초 내에 주제를 파악 가능해야 함.
    *   **오버레이 스타일:** 텍스트는 반드시 불투명도가 높은 블랙/화이트 박스를 사용하여 배경 이미지와 분리시켜야 하며, Primary Color(`FF4500`)는 **'긴급성(Urgency)'**을 나타내는 요소에만 제한적으로 사용한다.

### 2. 개발팀 전달용 최종 디자인 스펙 (Deliverables for Development Team)

개발팀은 아래 명시된 사양을 기반으로 실제 에셋을 구현해야 합니다.

#### A. 썸네일 디자인 스펙 상세 (Concept A 기반)

| 항목 | 스펙 상세 내용 | 기술적 요구사항 |
| :--- | :--- | :--- |
| **최종 해상도** | 1280x720 (YouTube 표준) 및 다양한 비율(1:1, 9:16) 지원 | PNG 또는 WebP 형식으로 제공. 크기별 최적화된 버전 필요. |
| **레이아웃 구조** | 중앙 집중형 비주얼 + 좌측 상단 핵심 문구 오버레이 | Figma 컴포넌트 기반으로 제작하여 재사용성 확보. |
| **색상 코드 (Hex)** | 배경: `#F5F5F5`, 텍스트 강조: `#FF4500`, 보조 텍스트: `#1A1A1A` | 모든 색상은 지정된 코드를 정확히 준수해야 함. |
| **타이포그래피** | 헤드라인: Montserrat Bold (최소 72pt), 본문: Noto Sans KR (최대 36pt) | 폰트 파일(Webfont 또는 SVG 경로 포함) 전달 필요. |
| **핵심 요소** | '강렬한 감정 유발 이미지'와 '명확한 문제/해결책 키워드'의 시각적 분리 명확화. | 오버레이 박스(Shadow 효과 포함)의 그림자 값(Box-shadow)은 `0 4px 8px rgba(0, 0, 0, 0.3)`을 표준으로 적용. |

#### B. 디자인 가이드라인 (Brand Style Guide Snippet)

개발팀이 향후 모든 콘텐츠에 일관성을 유지하도록 참조할 수 있는 핵심 규칙입니다.

1.  **톤앤매너 준수:** 시각적 요소는 **'전문성(Professionalism)'**과 **'도발성(Provocation)'** 사이의 균형을 유지해야 한다. 과도한 장식은 금지한다.
2.  **색상 사용 규칙:** Primary Color (`#FF4500`)는 **전체 콘텐츠 중 10% 미만**으로 사용하며, 오직 '클릭 유도 영역'에만 집중적으로 배치한다.
3.  **폰트 계층 구조:** 정보의 중요도 순서대로 크기와 굵기를 차등 적용하여 시각적 위계를 명확히 한다.

### 3. 최종 확인 및 개발 지시사항

개발팀은 상기 **A. 썸네일 디자인 스펙 상세**와 **B. 디자인 가이드라인**을 기준으로, 현재까지 승인된 모든 비주얼 에셋의 구현을 즉시 시작해야 합니다. 특히, 색상 코드와 타이포그래피 사용 규칙 준수를 최우선으로 확인한다.
